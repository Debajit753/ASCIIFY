02 — ABOUT

It’s a video player that thinks in characters.

ASCIIFY takes any video you give it and redraws every frame out of text — each character chosen by how bright that part of the image is, then painted in the exact color of the pixel underneath it.

The result plays like a normal video, with the original soundtrack, except the picture is made entirely of glyphs. It’s part toy, part homage to the amber and green CRT terminals this whole aesthetic comes from.

HOW IT WORKS

VIDEO ─▶ DOWNSCALE TO GRID ─▶ READ LUMINANCE ─▶ MATCH GLYPH ─▶ TINT W/ PIXEL ─▶ DRAW AUDIO ───────────────────────────────────────────────▶ PLAYS NATIVELY (MASTER CLOCK)
  1. 01Sample. Each frame is drawn onto a tiny hidden canvas, shrunk to the character grid — that grid size is the resolution.
  2. 02Measure. For every cell we read its pixel and compute brightness (Rec.709 luminance).
  3. 03Match. Brightness picks a character from the active ramp — dark → sparse, light → dense.
  4. 04Tint. The character is drawn in the source pixel’s real color (or green / inverted, depending on mode).
  5. 05Sync. Audio plays natively and acts as the clock; the visuals chase it and drop frames rather than drift.

COLOR MODES

COLOR

Each glyph keeps the true color of the pixel it replaced. The default, full-fidelity look.

MONO

Classic phosphor terminal — green on black, brightness only, no source color.

INVERT

A photographic negative: brightness and color of the color mode, flipped.

PRIVACY

Your video never leaves your device. There is no server and no upload — the file is read locally with the browser’s own APIs and processed frame-by-frame in the page. Close the tab and it’s gone. That’s the whole point of doing this client-side.

THE TECH

A hidden <video> decodes the file and supplies the audio. An offscreen <canvas> downsamples and reads pixels; a visible canvas draws the glyphs. The loop runs on requestAnimationFrame, keyed to the video’s own clock. No frameworks, no dependencies — just three static files: index.html, styles.css, app.js.

SOURCE

ASCIIFY is open source and ships as a few small static files with zero dependencies — clone it, open it, done. If it made you smile, a star is the nicest way to say thanks.
